### 即时通讯Demo

-----------

#### 简介
- 此Demo主要功能为即时通讯，实现了用户登录注册，单人、多人聊天，添加好友等功能。

------------

#### 提示
- 实现了手机号、用户名登录、注册。
- 发送和接收多媒体信息，一对一、讨论组、群、公共聊天室。
- 聊天界面对复杂类型消类型的处理，自动拉取历史消息
- 添加好友、加入群等。
- 此Demo基于腾讯IMSDK，由于GitHub有文件大小限制，现将第三方框架上传到百度云 [点击下载](https://pan.baidu.com/s/1bBNTAQ)
